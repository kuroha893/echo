# Guides

这个目录用于存放面向人类的高可读性说明文档。

这里的文档目标是：

- 帮新协作者快速理解 Echo 的架构
- 帮非实现者理解模块职责和系统边界
- 用比 protocol/spec 更易读的方式解释设计意图

这个目录不替代正式规范。

如果 `guides/` 中的解释性文档与正式规范冲突，请始终以以下内容为准：

1. `AGENTS.md`
2. `docs/governance/ai-engineering-constitution.md`
3. `docs/protocol/*.md`
4. `docs/runtime/*.md`

简单说：

- `docs/` 是 source of truth
- `guides/` 是帮助人快速理解 source of truth 的入口
